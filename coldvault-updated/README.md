## ColdVault (React + Tailwind)

### How to run

```bash
npm install
npm run dev
```

### Structure

- `src/App.jsx`: switches between Landing and Dashboard
- `src/pages/Landing/*`: landing page + hero visual
- `src/pages/Dashboard/*`: dashboard page + components
- `src/components/icons/*`: shared SVG icon set
- `src/styles/global.css`: Tailwind + your design tokens/animations/util classes
